# MemPhant Frame

## The Promise

Agents need memory that behaves like infrastructure, not a pile of embeddings.

MemPhant gives them:

- raw experience preserved
- useful memory derived
- scoped recall enforced
- poisoning resisted
- citations attached
- failures traceable
- evals reproducible

## The Public Line

MemPhant is the Apache-2.0 memory substrate for long-running agents.

One governed memory ledger, many surfaces: agents, RAG, coding, file-memory.

## The Launch Proof

Show a real trace:

```text
query -> candidates -> filters -> scores -> context -> citations -> answer
```

Then show the same query under ablations.

## The Boundary

MemPhant is not Syndai memory in public packaging. Syndai is the first production dogfood user.

## The Bet

The memory winner is not the project with the most knobs. It is the project that can prove which knob mattered.
